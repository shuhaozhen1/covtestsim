# Formal Simulation Results

This directory contains the lightweight formal simulation outputs that support
the manuscript:

- `summary/`: setting-level CSV summaries.
- `tables/`: manuscript/supplement TeX tables and same-stem CSV companions.
- `figures/`: manuscript power figures.
- `manifest.md`: reproducible commands and study inventory.
- `curated_latest_results_summary.md`: reader-oriented guide to the retained
  results.

Replicate-level raw files are intentionally omitted from the GitHub repository
because they are large. They can be regenerated with the commands in
`manifest.md`.
